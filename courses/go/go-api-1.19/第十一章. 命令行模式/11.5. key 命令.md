# 11.5. key 命令

原文链接：https://learnku.com/courses/go-api/1.19/key-command/13537

## 说明

这节课我们来开发 key 命令，生成随机字符串，可用来设置我们的 `APP_KEY` 环境变量的值。

## 1. 新建 key 命令

app/cmd/key.go

```
package cmd

import (
"gohub/pkg/console"
"gohub/pkg/helpers"

"github.com/spf13/cobra"
)

var CmdKey = &cobra.Command{
Use:   "key",
Short: "Generate App Key, will print the generated Key",
Run:   runKeyGenerate,
Args:  cobra.NoArgs, // 不允许传参
}

func runKeyGenerate(cmd *cobra.Command, args []string) {
console.Success("---")
console.Success("App Key:")
console.Success(helpers.RandomString(32))
console.Success("---")
console.Warning("please go to .env file to change the APP_KEY option")
}
```

## 2. 参数校验器

上面我们用到了 Args 参数校验：

```
Args:  cobra.NoArgs, // 不允许传参
```

这里来讲解下用法。

Cobra 提供了一组参数校验器，用以快速校验命令的参数是否符合预期，否则报错：

- `NoArgs` - 如果存在任何位置参数，该命令将报错

- `ArbitraryArgs` - 该命令会接受任何位置参数

- `OnlyValidArgs` - 如果有任何位置参数不在命令的 ValidArgs 字段中，该命令将报错

- `MinimumNArgs(int)` - 至少要有 N 个位置参数，否则报错

- `MaximumNArgs(int)` - 如果位置参数超过 N 个将报错

- `ExactArgs(int)` - 必须有 N 个位置参数，否则报错

- `ExactValidArgs(int) 必须有 N 个位置参数，且都在命令的 ValidArgs 字段中，否则报错

- `RangeArgs(min, max)` - 如果位置参数的个数不在区间 min 和 max 之中，报错

- `MatchAll(pargs ...PositionalArgs)` -  支持使用以上的多个验证器

>

注意: 参数（args）不是选项（flags），可以查看这里 [learnku.com/courses/go-api/1.17/co...](https://learnku.com/courses/go-api/1.17/command-line-basics#05fbe6) 加深印象。

默认报错时，会报错时会打印当前命令的帮助信息。

## 3. `helpers.RandomString()`

生成随机字符串的函数，可能程序里的别的地方也会用到，我们放到 helpers 库里：

pkg/helpers/helpers.go

```

import (
.
.
.
mathrand "math/rand"

// RandomString 生成长度为 length 的随机字符串
func RandomString(length int) string {
mathrand.Seed(time.Now().UnixNano())
letters := "abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ"
b := make([]byte, length)
for i := range b {
b[i] = letters[mathrand.Intn(len(letters))]
}
return string(b)
}
```

注意顶部的 `mathrand "math/rand"`，因为我们已经加载了 `"crypto/rand"`，为避免 import 冲突，使用 mathrand 作为别名。

## 5. 测试

测试之前我们来将日志等级设置为 error，让终端输出保持干净：

.env

```
LOG_LEVEL=error
```

运行命令：

```
$ go run main.go key
---
App Key:
nooNSQIBSidfqRztvZIdOzwloOFWQAYS
---
please go to .env file to change the APP_KEY option
```

我们上面设置了不允许参数，试着传参：

```
$ go run main.go key test
Error: unknown command "test" for " key"
Usage:
key [flags]

Flags:
-h, --help   help for key

Global Flags:
-e, --env string   load .env file, example: --env=testing will use .env.testing file
```

因为我们不允许传参，所以 cobra 会去尝试子命令，最终打印出来以上错误和帮助信息。

一切符合预期。
