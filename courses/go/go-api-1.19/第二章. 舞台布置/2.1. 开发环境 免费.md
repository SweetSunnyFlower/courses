## 其他软件

### Redis 管理工具

推荐使用 [AnotherRedisDesktopManager](https://github.com/qishibo/AnotherRedisDesktopManager/releases)，跨平台，支持 Mac、Windows 和 Linux。

### 数据库管理工具

推荐使用 [TablePlus](https://tableplus.com/) ，跨平台，支持 Mac、Windows 和 Linux。

另外，TablePlus 除了支持 MySQL 以外，还支持 Redis / Sqlite / PostgreSQL  / MariaDB / Cassandra / MongoDB 和一堆其他的数据库。
