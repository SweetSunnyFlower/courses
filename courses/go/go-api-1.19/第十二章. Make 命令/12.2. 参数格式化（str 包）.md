# 12.2. 参数格式化（str 包）

原文链接：https://learnku.com/courses/go-api/1.19/parameter-formatting-str-package/13541

## 说明

本节课我们来创建 str 包，这个用来格式化用户参数。

## 1. 什么是参数格式化？

例如 make model 命令指纹是：

```bash
$ gohub make model {模型名称}
```

参数 `{模型名称}` 用户可能会传进来各种奇奇怪怪的名称，如要创建 TopicComment 模型文件，是用户有可能传进来的参数：

- topic_comments

- TopicComment

- TopicComments

- topicComment

- topicComments

- topic_comment

所以我们需要有个 str 包，来对就用户输入的内容进行格式化处理。

## 2. 安装依赖

开始之前我们先来按照依赖包。

[strcase](https://github.com/iancoleman/strcase) 是用来处理大小写：

```bash
$ go get github.com/iancoleman/strcase
```

[go-pluralize](https://github.com/gertd/go-pluralize) 用来处理英文单复数：

```bash
$ go get github.com/gertd/go-pluralize
```

## 3. 创建 str 包

pkg/str/str.go

```go
// Package str 字符串辅助方法
package str

import (
	"github.com/gertd/go-pluralize"
	"github.com/iancoleman/strcase"
)

// Plural 转为复数 user -> users
func Plural(word string) string {
	return pluralize.NewClient().Plural(word)
}

// Singular 转为单数 users -> user
func Singular(word string) string {
	return pluralize.NewClient().Singular(word)
}

// Snake 转为 snake_case，如 TopicComment -> topic_comment
func Snake(s string) string {
	return strcase.ToSnake(s)
}

// Camel 转为 CamelCase，如 topic_comment -> TopicComment
func Camel(s string) string {
	return strcase.ToCamel(s)
}

// LowerCamel 转为 lowerCamelCase，如 TopicComment -> topicComment
func LowerCamel(s string) string {
	return strcase.ToLowerCamel(s)
}
```
