# 17.7. 总结

原文链接：https://learnku.com/courses/go-api/1.19/summary/13588

## 本章知识点：

- 创建友情链接模型；

- 创建友情链接迁移；

- 创建友情链接数据工厂；

- 创建友情链接数据填充；

- 缓存包的设计；

- 使用 Go interface 来做缓存存储的抽象化；

- 缓存与业务逻辑使用不同数据库；

- `link.AllCached()` 优先加载缓存；

- cache clear 命令，清空缓存；

- cache forget 命令，删除单条缓存。
