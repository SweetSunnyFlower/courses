# 14.3. embed 打包 CSS/JS 静态文件

原文链接：https://learnku.com/courses/go-basic/1.22/package-css-js-static-files-with-embedded/16560

## 说明

上一节我们打包了模板文件，这节课演示如何打包 CSS / JS 静态文件。

## 开始编码

同样的，我们也需要在 main 中创建加载静态文件的 `//go:embed` 指令，并将变量传参到路由中：

main.go

```go
package main

import (
	"embed"
	"goblog/app/http/middlewares"
	"goblog/bootstrap"
	"goblog/config"
	c "goblog/pkg/config"
	"goblog/pkg/logger"
	"net/http"
)

//go:embed resources/views/articles/*
//go:embed resources/views/auth/*
//go:embed resources/views/categories/*
//go:embed resources/views/layouts/*
var tplFS embed.FS

//go:embed public/*
var staticFS embed.FS

func init() {
	// 初始化配置信息
	config.Initialize()
}

func main() {
	// 初始化 SQL
	bootstrap.SetupDB()

	// 初始化模板加载
	bootstrap.SetupTemplate(tplFS)

	// 初始化路由绑定
	router := bootstrap.SetupRoute(staticFS)

	err := http.ListenAndServe(":"+c.GetString("app.port"), middlewares.RemoveTrailingSlash(router))
	logger.LogError(err)
}
```

需要修改 `bootstrap.SetupRoute` 方法，增加参数 staticFS ：

bootstrap/route.go

```go
// Package bootstrap 负责应用初始化相关工作，比如初始化路由。
package bootstrap

import (
	"embed"
	"goblog/pkg/route"
	"goblog/routes"
	"io/fs"
	"net/http"

	"github.com/gorilla/mux"
)

// SetupRoute 路由初始化
func SetupRoute(staticFS embed.FS) *mux.Router {
	router := mux.NewRouter()
	routes.RegisterWebRoutes(router)

	route.SetRoute(router)

	// 静态资源
	sub, _ := fs.Sub(staticFS, "public")
	router.PathPrefix("/").Handler(http.FileServer(http.FS(sub)))

	return router
}
```

我们可以将静态路由的设定放到 SetupRoute 方法里，没必要再将 staticFS 变量再传参到 routes.RegisterWebRoutes 方法里。

进行子目录读取操作：

```
sub, _ := fs.Sub(staticFS, "public")
```

相当于去除了 public 前缀，否则 URL 将变成以下这样才能正确读取：

```
http://localhost:3000/public/css/app.css
```

而我们的项目里需要的是：

```
http://localhost:3000/css/app.css
```

最后搭配 `http.FS(sub)` 即可完成将 `/` 匹对到 `public` 目录下：

```
router.PathPrefix("/").Handler(http.FileServer(http.FS(sub)))
```

## 删除无用路由

进入 RegisterWebRoutes 中，删除以下两行静态资源路由的代码：

```
// 静态资源
r.PathPrefix("/css/").Handler(http.FileServer(http.Dir("./public")))
r.PathPrefix("/js/").Handler(http.FileServer(http.Dir("./public")))
```

## 测试一下

运行：

```bash
$ go build -o goblog
```

将 public 目录删除，手动删除或者使用以下命令：

```bash
$ rm -rf public
```

运行我们的 Goblog 程序：

```bash
$ ./goblog
```

打开浏览器访问我们的应用 [localhost:3000](http://localhost:3000/) 可以看到样式正常，这意味着我们的 goblog 是从二进制的 embed 文件系统中加载了 CSS 和 JS 文件。

最后将我们上面删除的目录还原回来：

```bash
$ git checkout public
```
